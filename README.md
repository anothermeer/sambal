# sambal
> An easy-to-use tool for sharing files and text across devices using local network.  

# Building Requirements
Go
Cobra
Fyne

# Installation
Windows:
```powershell
git clone https://github.com/anothermeer/sambal
./build_all.bat
```
Linux/Unix:
```shell
git clone https://github.com/anothermeer/sambal
./build_all.sh
```
